NEW: Learning R

Below are materials for 6 lessons introducing you to R and RStudio with a heavy focus on Tidyverse packages. The lessons are designed to be used in order, with each gradually building on the others. The last two sessions are a bit more specialized, showing you how to access the U.S. Census Bureau API using TidyCensus and how to turn your R code into an HTML page using RMarkdown.

How to get started

  • Download R. It will ask you to pick your CRAN mirror. Just pick one geographically close to you.
  • Download RStudio. The free version is sufficient.
  • For the Using Tidycensus class, you will need to get a Census API key.
  • Read through the Pros and Cons of R.
  • Bookmark this R Cheat Sheet for future reference.
  • If you'd like a small taste of R before launching into these classes, check out this gentle introduction via a web page
  • Download this R script file called "Setting up for R Classes" and save it to your computer in a place you can find it later.
  • Open RStudio and go to the File menu, choose "Open file" and navigate to the "Setting up for R classes" file that you just downloaded.
  • Follow the directions in the R script file. Note: you will need to change a file path in section 2 of the code. This script will create directories on your computer, download and unzip the relevant materials for each of the six classes.
  • When the script is finished, close the script file. If you're ready to jump into the classes, keep RStudio open and go to the next step.
  • I've also started creating some videos to go with these. Links are below if there's a video.

The six classes

  • To launch a class: In RStudio, go to the file menu and choose "New Project." In the pop-up window that comes up, choose "Existing directory" and navigate to the folders that the R Script just made for you. Choose the folder that corresponds to the class you want to do and then push "Create Project". This will create a new object in that folder (ending in .Rproj). It essentially sets up the working environment for you. (You don't have to use projects to work in R, but it's highly recommended). Next, look in the lower right corner of RStudio and click on the "Files" tab. You will see the files that were downloaded for you. Below is a list of the first file to open for each class. Then follow the directions in that file to continue in the class. Repeat this whole process for each class. I've set up each class as a separate project in order to give you repetitive practice in setting up a project.
  • Class 1: Getting started with R. First file to open: "GettingStartedWithRScript.R"
  • Class 2: Using Tidyverse, part 1. "TidyversePart1.rmd"
  • Class 3: Cleaning data. "PowerofDataCleaning.rmd"
  • Class 4: Using Tidyverse, part 2. First file to open: "vaxx_import.R"
  • Class 5: Using Tidycensus. "UsingTidyCensus.rmd". Also download a zip file with Part 1 videoand a second zip file with Part 2 video
  • Class 6: Using RMarkdown. "R_Markdown.rmd"
  • When you are done with a class, go to the File menu and choose "Close Project" and be sure that everything saves. You can return to a project by opening RStudio and choosing "Open Project."

SQLite using Firefox SQLite Manager

SQLite in Firefox is a light-weight SQL tool, but it has its limitations. The best option would be to set up your own instance of mySQL, but if that's beyond your capabilities, this would at least give you an introduction to Structured Query Language.

  • Make sure you have an updated version of Firefox installed on your computer, then download the SQLite Manager Firefox Add-on and that's all you need to get started! More specific directions are in the first exercise below.
